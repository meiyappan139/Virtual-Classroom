Interview Process

         
         Interview Rounds

          Online Round

          There were around 20 MCQs ( No technical questions ).
          There were 2 Einstein puzzles.
          Other questions included few logical and few quantitative aptitude questions.
          The time given was 45 minutes.
          The next section had two coding questions out of which one is to be answered. The questions were
              • Given an array of n elements, rotate k elements to the left or right based on the flag value
              • Finding minimum edit distance in a string

          Technical Round 1

          1. The interviewer asked me to explain the code which was asked during the first round and trace it with an example.
          2. Given an Arabic number write a code to convert it to corresponding roman number.

          Technical Round 2

          1. As my project was based on Graph there were few questions on Graphs.
          2. Then he asked me few questions on arrays,linked list and trees.
          3. In DBMS he mainly focused on ER model . I was given a scenario and was asked to design a ER as well as Relational model for the same.

          HR Round

        • Strengths and weakness
        • How will I face criticisms and asked me to state 2 such situations
        • Secret behind my consistent performance in academics
        • Leadership qualities
        • Few scenario based questions
        • Why PayPal
        • Why should we hire you and what makes you different from others
